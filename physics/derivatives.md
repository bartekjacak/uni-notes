# derivatives

### notation

For $f(t)$ first and second derivative:

- $\frac{df}{dt}$, $\frac{d^2f}{dt^2}$
- $f'(t)$, $f''(t)$
- $\dot{f}$, $\ddot{f}$

### rules

| function            | derivative with respect to $x$         |
| ------------------- | -------------------------------------- |
| $Ax^B$              | $ABx^{B-1}$                            |
| $f(x)g(x)$          | $f'(x)g(x) + f(x)g'(x)$                |
| $\frac{f(x)}{g(x)}$ | $\frac{f'(x)g(x) - f(x)g'(x)}{g^2(x)}$ |
| $\sin{x}$           | $\cos{x}$                              |
| $\cos{x}$           | $-\sin{x}$                             |
| $f(g(x))$           | $f'(g(x))g'(x)$                        |
| $e^x$               | $e^x$                                  |
