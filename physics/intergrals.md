# integrals

### notation

$\int f'(x) dx = f(x) + const$

### rules

| function     | integration with respect to $x$ |
| ------------ | ------------------------------- |
| $a$          | $ax + const$                    |
| $x^n$        | $\frac{1}{n+1} x^{n+1} + const$ |
| $e^x$        | $e^x + const$                   |
| $a^x$        | $\frac{a^x}{\ln(a)} + const$    |
| $\ln(x)$     | $x\ln(x) - x + const$           |
| $\sin(x)$    | $-\cos(x) + const$              |
| $\cos(x)$    | $\sin(x) + const$               |
| $\int af(x)$ | $a \int f(x)$                   |
