# kinematics

Study of motion.

### reference

| variable | meaning                                   |
| -------- | ----------------------------------------- |
| $a$      | acceleration                              |
| $v$      | velocity                                  |
| $x,y,z$  | position                                  |
| $x,y,z$  | position                                  |
| $g$      | gravitation ($\approx 9.8 \frac{m}{s^2}$) |

#### averages

$v_{av} = \bar{v} = \frac{\Delta x}{\Delta t}$

$a_{av} = \bar{a} = \frac{\Delta v}{\Delta t}$

#### movement formulas

$s(t) = s_{0} + v_{0}*t + \frac{a*t^2}{2}$

$s'(t) = v(t) = v_{0} + a*t$

$s''(t) = v'(t) = a(t) = a$
